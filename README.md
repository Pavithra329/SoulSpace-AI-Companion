# 🌿 SoulSpace AI Companion

SoulSpace is a personalized mental wellness companion designed to provide a safe and supportive space for users to express their thoughts, track emotions, and reflect through journaling.

---

## 💬 Features

### 🧠 Emotion-Aware Chat
- Detects user emotions from input text
- Provides short, supportive, and human-like responses
- Uses a safe rule-based system for reliability

### 💙 Motivational Support
- Generates personalized motivational messages based on mood
- Encourages positive thinking and emotional balance

### 📊 Mood Tracking
- Records user moods over time
- Displays mood trends using simple visualizations

### 📝 Guided Journaling
- Provides mood-based prompts
- Allows users to reflect and store thoughts during the session
